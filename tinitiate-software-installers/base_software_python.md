# 🐍 Python & Base Software Installer

> **Use this to install Python and essential base software on your Windows system. Also use wherever Python is required.**

## 📥 Step 1: Install

Follow these steps to install all the required software:

1. **Click on the Windows Start button**
2. **Type "Windows PowerShell" in the search bar**
3. **Look for the Windows PowerShell in the search results**
4. **Right click on it and select "Run as administrator"**
5. **Copy the following command and run it:**

```powershell
iex (Invoke-WebRequest -Uri "https://raw.githubusercontent.com/tinitiateprime/tinitiate-onboarding/main/software-installers/windows/python-installers/python_installers.ps1" -UseBasicParsing).Content
```

6. **Wait until everything gets installed**
7. **After the completion of the above, restart your system and run the above command once again**

## ✅ Step 2: Check Installation

After restarting your system, verify that the following software has been installed:

### 📦 Software Packages
- **Chocolatey** (The software app store)
- **Python** (Programming language)
- **Notepad++** (Text editor)
- **Git** (Version control system)
- **VS Code** (Code editor)
- **DBeaver Community** (Database management tool)
- **MS Teams (Work or School)** (Collaboration platform)
- **Zoom** (Video conferencing)

### 🔍 How to Check
- Look for these applications in your Windows Start menu
- Or check if you can run them from the command line (e.g., `python --version`, `git --version`, `code --version`)

## ⚠️ Important Notes
- Make sure to run PowerShell as Administrator
- The installation process may take several minutes
- A system restart is required after the first installation
- Run the command again after restart to complete setup
