# Python Developer Program

Use this setup for Python basics and MySQL practice.

## YAML File
[python_only.yml](./python_only.yml)

## Start Command

```powershell
docker compose -f python_only.yml up -d
```

## Software Included

* 📓 **Jupyter**: http://localhost:8888
* 🐬 **MySQL**: `localhost:3306`

## Jupyter Token

```
admin123
```

## Tutorials Included

* [Python](https://github.com/tinitiateprime/python.git)

## Tutorial Location

Tutorial repositories are downloaded into:

```text
./notebooks/tutorials
```

## To Stop the Setup

```powershell
docker compose -f python_only.yml down
```
